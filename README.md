# Destination Change Unified Flow

This package reverts the app to the pre-vendor-update logic and adds a fallback equalization pass for items where Safety Stock = 0.

## Files
- destination_change_streamlit_app.py
- destination_change_unified_flow.py
- requirements.txt

## Run locally
```bash
pip install -r requirements.txt
streamlit run destination_change_streamlit_app.py
```
